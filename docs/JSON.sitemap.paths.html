Class: JSON.sitemap.paths

.sitemap. paths

Sitemap.json Paths Class

The "paths" object contains any number of named path arrays, each of which lists (in order) a list of page IDs making a swipe navigation path.

"default" path is required, but the name may be changed via eDetailer.SETTINGS.primary

Source:
sitemap.sample.js, line 270

Properties:

Name Type Argument Description
default {array} - Primary swipe path
<name> {array} optional
 - Additional named swipe paths
