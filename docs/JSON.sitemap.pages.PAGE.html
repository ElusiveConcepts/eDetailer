Class: JSON.sitemap.pages.PAGE

.sitemap.pages. PAGE

Sitemap.json Page Class

desc
Source:
sitemap.sample.js, line 56

Properties:

Name Type Argument Description
id {string} - ID used to reference the page (also used for tracking)
title {string} - Title of the page (used in navigation and tracking)
type {String} - Type of page/sitemap entry [page, hidden, header, button]
  • page : Normal page/slide, appears in all menus
  • hidden : normal page/slide, but does not appear in menus
  • header : Accordian or Section header, appears in most menus
  • button : Appears in some menus, file content is placed in HREF
file {string} optional
 - Path and filename of the source code for the page, or HREF content in the case of buttons
children {array} optional
 - An array of child (nested) page objects, no depth limit

Examples

Create an accordian menu header

{
	"id"    : "header.0.0.0",
	"file"  : "", // Not necessary as this doesn't link to a page
	"title" : "Section Header",
	"type"  : "header",
}

Create a standard page entry (visible in menus)

{
	"id"    : "slide.0.0.0",
	"file"  : "slides/slide.0.0.0.html",
	"title" : "Slide Title",
	"type"  : "page"
}

Create a standard page entry with children

{
	"id"    : "slide.1.0.0",
	"file"  : "slides/slide.1.0.0.html",
	"title" : "Slide Title",
	"type"  : "page",
	"children" : [
		{
			"id"    : "slide.1.1.0",
			"file"  : "slides/slide.1.1.0.html",
			"title" : "Sub Slide Title",
			"type"  : "page"
		}
	]
}

Create a hidden page entry (not visible in menus)

{
	"id"    : "splash",
	"file"  : "slides/splash.html",
	"title" : "Splash Page",
	"type"  : "hidden"
}

Create a button

{
	"id"    : "REFS",
	"file"  : "javascript:eDetailer.LIGHTBOX.show('popups/refs.html', 'ajax');",
	"title" : "References",
	"type"  : "button"
}
