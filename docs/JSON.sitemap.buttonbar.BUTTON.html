Class: JSON.sitemap.buttonbar.BUTTON

.sitemap.buttonbar. BUTTON

Sitemap.json Button Class

desc
Source:
sitemap.sample.js, line 164

Properties:

Name Type Argument Description
id {string} - ID used to reference the button
file {string} optional
 - HREF content (path, link, or JavaScript)
title {string} - Title of the button
type {string} - Type of entry [button]
  • button : Appears in some menus, file content is placed in HREF

Examples

Describe a button to load a slide

{
	"id"    : "Home",
	"file"  : "javascript:eDetailer.PAGE.load('slide.0.0.0', 'B2T');",
	"title" : "Home",
	"type"  : "button"
}

Describe a button to display a lightbox

{
	"id"    : "REFS",
	"file"  : "javascript:eDetailer.LIGHTBOX.show('popups/refs.html', 'ajax');",
	"title" : "References",
	"type"  : "button"
}

Describe a button to load a local file

{
	"id"    : "PI",
	"file"  : "pdfs/prescribing_information.pdf",
	"title" : "Prescribing Information",
	"type"  : "button"
}

Describe a button to load an external site

{
	"id"    : "example_com",
	"file"  : "http://www.example.com",
	"title" : "example.com",
	"type"  : "button"
}
